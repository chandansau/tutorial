Array Methods: 
    ar=array.array('i',[2,5,8,-6,3,4,9])

    ar.append(x) :   This function is used to add the value mentioned in its arguments at the end of the array.
    ar.count(x):     Returns the number of occurrances of x in the array a.
    ar.insert(i,x) : This function is used to add the value at the position specified in its argument.
    ar.index(x):     Return the position number of first occurrence of x'
    ar.pop(x):       Remove the the item of x from array and return it.
    ar.pop():        Remove last item from array.
    ar.remove(x) :   Return first occrunce of x in the array.
    ar.reverse():    Reverse the order of the array.

    Example
    #effect of Methodes
    import array as a
    ar=a.array('i',[2,5,8,-6,3,4,9])

Creating array using linspace() 
    syntax:
    linspace(start:stop:n)
    n= divided start to stop into n part and take those point in the array.
    Example
#creating a array using linspace(start,end,part)

    from numpy import *
    ar=linspace(0,10,5)
    print(ar)
    o/p:
    [  0.    2.5   5.    7.5  10. ]
Creating array using logspace() 
    syntax:
    logspace(start:stop:n)
    This function start at a value which is start to the power of 10 and ends at a value which is stop to the power of 10.
    Example
#creating a array using logspace(start,end,part)

    from numpy import *
    ar=logspace(1,4,5)
    print(ar)
    o/p:
    [    10.             56.23413252    316.22776602   1778.27941004  10000.        ]

List Comprehension:
    Creating a new list from a inerrable object
    Syntax:
    [expression for item in list if conditional]
    Example
    #create a list comprehension with squre of 1 to 10
    _________________________________________________________
    1.
    lst=[]
    for i in range(1,11):
        lst.append(i**2)
    print(lst)
    ----------------------------------------------------
    2.
    lst=[pow(i,2) for i in range(1,11)]
    print(lst)
    ________________________________________________________
    same o/p for 1 and 2
    o/p:
    [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
    ________________________________________________________
    # Even Numbers
    lst=[i for i in range(1,11) if(i%2==0)]
    print(lst)
    o/p:
    [2, 4, 6, 8, 10]
