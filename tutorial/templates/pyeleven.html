Polymorphism is an important feature of class definition in Python that is utilized when you have commonly named methods across classes or subclasses.
Poly means many and morphis means many forms. It something exhibits various form called polymorphism. Advantages of Polymorphism

=>It helps the programmer to reuse the codes, i.e., classes once written, tested and implemented can be reused as required. Saves a lot of time.

=>Single variable can be used to store multiple data types.

=>Easy to debug the codes.

Duck typing philosophy of python
 calling a method without knowing the type(class)  of the object.

    Example
    #duck type
    class Duck:
        def talk(self):
            print("Quack,quack!")
    class Dog:
        def talk(self):
            print("Bow,wow!!")
    #duck typing
    def callit(obj):
        obj.talk()
    obj1=Duck()
    callit(obj1)

    obj2=Dog()
    callit(obj2)

    o/p:
    Quack,quack!
    Bow,wow!!

Method overloading

Method overriding
Override means having two methods with the same name but doing different tasks. It means that one of the methods overrides the other.
If there is any method in the superclass and a method with the same name in a subclass, then by executing the method, the method of the corresponding
class will be executed.

    Example
    #derive rectangle class from squre class and display area of area of 2 class
    class Sqr:
        def area(self,x,y):
            print("area of rec:\t",x+y)
    class Rec(Sqr):
        def area(self,x,y):
            print("area of Rec:\t",x*y)

    r=Rec()
    r.area(6,4);
    o/p:
    area of Rec:	 24


Abstract class
    ïƒ¨	It is class that contain 0 or more abstract method. It have no body
    Example
       # create a abstract class with an abstract method to calculate squre ,sqrt ,cube value
    from abc import *
    import math as m
    class Myclass(ABC):
        @abstractmethod
        def calculate(self,x):
            pass
    class sub1(Myclass):
        def calculate(self,x):
            return(pow(x,2))

    class sub2(Myclass):
        def calculate(self,x):
            return(x*x*x)
    class sub3(Myclass):
        def calculate(self,x):
            return(m.sqrt(x))
    ob=sub1();
    ob1=sub2();
    ob2=sub3();
    print(ob.calculate(9))
    print(ob1.calculate(9))
    print(ob2.calculate(9))
